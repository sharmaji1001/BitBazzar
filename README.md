# BitBazzar
BitBazaar is a web application designed to help users track their progress in completing daily coding challenges on platforms like **GeeksforGeeks (GFG)**. It provides a simple interface where users can input their current streak, current bonus, target streak, and current Geekbits balance. Based on this input, BitBazaar calculates the remaining days needed for the user to achieve their target streak and earn bonus Geekbits. The application aims to motivate users to stay consistent with their daily coding practice by providing them with a clear goal and tracking their progress towards it. Through its user-friendly design and functionality, BitBazaar serves as a valuable tool for coding enthusiasts looking to improve their skills and earn rewards on coding challenge platforms.
